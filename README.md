# 📊 Proyecto Integrador: Visualización de Datos con Power BI (Módulo 8)
### Diplomado en Fundamentos de Análisis de Datos — SENCE / Talento Digital

## 📝 Descripción
Este proyecto corresponde al **Módulo 8** y se enfoca en la creación de un **Dashboard interactivo** utilizando **Power BI Desktop**. El objetivo es transformar los datos crudos del eCommerce en información visual estratégica para la toma de decisiones.

## 🎯 Objetivos Técnicos
- **ETL con Power Query:** Limpieza de datos, corrección de tipos de datos mediante configuración regional y tratamiento de decimales.
- **Modelamiento DAX:** Creación de medidas calculadas para KPIs principales (Venta Total, Promedio de Gasto, Total de Clientes, Promedio de Edad).
- **Diseño de Dashboard:** Implementación de visualizaciones interactivas con formatos de moneda y separadores de miles.

## 📊 Análisis del Dashboard
El tablero presenta 4 pilares fundamentales:
1. **KPIs Principales:** Resumen ejecutivo de ventas, gastos y clientes con formato financiero.
2. **Ventas por Ciudad:** Identificación de Santiago de Chile como el mercado líder.
3. **Segmentación de Clientes:** Distribución porcentual entre clientes Premium y Regulares.
4. **Comportamiento por Edad:** Análisis de la relación entre el rango etario y el volumen de compra.

## ✅ Conclusión
La implementación de este Dashboard permite una visualización inmediata de la salud del negocio. Gracias a la interactividad de Power BI, es posible filtrar por ciudad o categoría de cliente para obtener insights específicos, facilitando una gestión basada en datos reales.
